# Car-Price-Prediction
Machine Learning project for predicting Fiat 500 car prices using Python. Includes data preprocessing, EDA, Linear Regression, Ridge Regression, Random Forest, hyperparameter tuning, and performance evaluation.
# 🚗 Car Price Prediction using Machine Learning

## 📌 Overview

This project predicts the price of Fiat 500 cars using Machine Learning techniques. It includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and performance evaluation using multiple regression algorithms.

---

## 🎯 Objectives

- Analyze the car dataset
- Perform Exploratory Data Analysis (EDA)
- Preprocess and clean the data
- Train multiple regression models
- Compare model performance
- Predict car prices for new data

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## 📂 Dataset

The project uses the **Fiat 500** dataset containing features such as:

- Engine Power
- Age of Vehicle
- Kilometers Driven
- Previous Owners
- Model
- Latitude & Longitude
- Price (Target Variable)

---

## 📊 Exploratory Data Analysis

The project includes:

- Dataset summary
- Missing value analysis
- Correlation Heatmap
- Line plots
- Scatter plots
- Bar charts

---

## 🤖 Machine Learning Models

The following algorithms were implemented:

- Linear Regression
- Ridge Regression
- Random Forest Regressor

Hyperparameter tuning is performed using **GridSearchCV**.

---

## 📈 Model Evaluation

The models are evaluated using:

- R² Score
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
